This program runs from the command line.
The default settings are to run a graph search witht he first Heuristic.
In order to run a Tree search add the argument --tree
In order to run the second Heuristic add the argument --h2

In order to run a custom puzzle, simple enter the numbers in order the order of your puzzle from left to right
top to bottom, each nmumber seperated by a space.
Use 0 as a place holder for the empty space.

First, compile the Driver.java file.
Next run the Driver.class file.
This file can run without any additional arguments, for default setting
or any of the three additional arguments

An example of a custom exicution would be "java Driver.class 1 0 2 3 4 5 6 7 8 --tree --h2"
